# GitHub Project Management

## 📌 Project Description
GitHub Project Management is a Java Maven project created to learn GitHub collaboration and project management practices. It demonstrates repository management, branching strategies, pull requests, code reviews, issue tracking, project boards, and release management.

---

## 🚀 Technologies Used

- Java 17
- Maven
- Git
- GitHub
- VS Code

## ⚙️ Setup Instructions

### Clone Repository

```bash
git clone https://github.com/yourusername/github-project-management.git
```

### Open Project

```bash
cd github-project-management
```

### Compile Project

```bash
mvn clean compile
```

### Package Project

```bash
mvn package
```
